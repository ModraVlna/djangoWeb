Funkce

Funkce je blok kódu, která slouží pro vykonání nějaké činnosti. Funkce umožňuje opakovaně použít blok kódu, aniž bychom ho museli pokaždé znovu opisovat. Funkce se skládá z hlavičky a z těla. Hlavička funkce obsahuje na začátku klíčové slovo def, poté následuje název funkce, závorky a dvojtečka. V závorce mohou a nemusí být parametry. Za hlavičkou následuje tělo. Tělo obsahuje příkazy, které budou postupně vykonány po zavolání funkce. Každý nový řádek je odsazen tabulátorem.

def moje_funkce(): # hlavicka
    print("Ahoj") # telo

def moje_funkce(): # hlavicka
print("Ahoj") # telo


Vytvoření a práce s vlastní funkcí

Bude funkce s názvem pozdrav, která vypíše na výstup nějaký pozdrav.

def pozdrav(): # hlavicka
    print("Ahoj") # telo

def pozdrav(): # hlavicka
print("Ahoj") # telo


Aby příkaz uvnitř funkce byl proveden, je potřeba funkci zavolat. Funkce se zavolá tak, že se napíše její název a závorky.

def pozdrav():
    print("Ahoj")

pozdrav() # vypise: Ahoj

def pozdrav():
print("Ahoj")

pozdrav() # vypise: Ahoj


Tato funkce pozdrav(), má prázdné závorky, neobsahuje tedy žádné parametry. Parametr lze představit jako data, která bude daná funkce nějakým způsobem požívat. V naší funkci pozdrav, přidáme parametr s názvem jmeno.

def pozdrav(jmeno):
    print("Ahoj")

def pozdrav(jmeno):
print("Ahoj")


Uvnitř kulatých závorek je tedy parametr jmeno. Tato funkce nyní vypíše náš pozdrav a bude za ní následovat jméno uživatele.

def pozdrav(jmeno):
    print("Ahoj " + jmeno)

def pozdrav(jmeno):
print("Ahoj " + jmeno)


Zase je potřeba funkci zavolat. Jelikož funkce má definován parametr, tak při volání se musí do závorek přidat nějaká hodnota. Hodnota, které se přidává při volání funkce do závorek se označuje jako argument.

def pozdrav(jmeno):
    print("Ahoj " + jmeno)

pozdrav("Gustav") # vypise: Ahoj Gustav

def pozdrav(jmeno):
print("Ahoj " + jmeno)

pozdrav("Gustav") # vypise: Ahoj Gustav


Funkce může mít více parametrů. Při volání funkce a zadávání argumentů je potřeba, aby pořadí bylo zachováno. Počet parametrů a argumentů se musí vždy u dané funkci rovnat.

Vytvoříme funkci, která bude sčítat dvě hodnoty. Funkce bude mít dva parametry, které představují dvě hodnoty. Funkce vypíše na výstup výsledek.

def secti(a, b):
    print(a+b)

secti(6, 10) # vypise: 16

def secti(a, b):
print(a+b)

secti(6, 10) # vypise: 16


Příkaz return

Příkaz return se používá v rámci funkce pro vrácení nějaké hodnoty. Máme příklad, kde sčítáme dvě hodnoty. Výsledek sečtení můžeme vypsat na výstup nebo vrátit. Výpis hodnoty na výstup se dá provést pomocí print(). Vrácení hodnoty je pomocí příkazu return. Příkaz return se napíše uvnitř těla funkce před hodnotou, kterou chceme vypsat.

def secti(a, b):
    return a+b

secti(6, 10)

def secti(a, b):
return a+b

secti(6, 10)


Častá je otázka, jaký je rozdíl mezi použitím print() a return.
return umožňuje uložit výsledek jako proměnnou a dále s ní pracovat.

Ukážeme si to na příkladě. Vytvoříme dvě funkce, kde porovnáme použití print() a return. První funkce se bude jmenovat return_vysledek(). Druhá funkce se bude jmenovat print_vysledek(). Funkce sečte dvě hodnoty. Zavoláme tuto funkci. Výsledek sečtení přiřadíme k proměnné s názvem vysledek a přičteme hodnotu 100.

První funkce tedy vrací součet dvou hodnot a přiřadí je k proměnné a přičteme ještě hodnotu 100.

def return_vysledek(a, b):
    return a+b # 6 + 10

vysledek = return_vysledek(6, 10) + 100 # 16 + 100
print(vysledek) # vypise: 116

def return_vysledek(a, b):
return a+b # 6 + 10

vysledek = return_vysledek(6, 10) + 100 # 16 + 100
print(vysledek) # vypise: 116


Druhá funkce bude stejná, ale místo klíčového slova return, bude mít v těle print()

def print_vysledek(a, b):
    print(a+b)

vysledek2 = print_vysledek(6, 10) + 100 # Vyhodi chybu!

def print_vysledek(a, b):
print(a+b)

vysledek2 = print_vysledek(6, 10) + 100 # Vyhodi chybu!


Použití druhé funkce vyhodí chybu. Protože proměnná vysledek2 je typu NoneType. print() nevrací žádnou hodnotu, která by se dala ukládat. Z tohoto důvodu nelze k vysledek2 přičítat další hodnoty.

Práce s funkcí

Vytvoříme funkci, která bude rozhodovat o tom, zda číslo je nebo není sudé. Pokud číslo je sudé, funkce vrátí True. Pokud číslo není sudé, funkce vrátí False. Funkce bude mít jeden parametr, který bude představovat zadané číslo.

def sude_cislo(cislo):
    vysledek = cislo % 2 == 0
    return vysledek

print(sude_cislo(10)) # vypise: True
print(sude_cislo(11)) # vypise: False

def sude_cislo(cislo):
vysledek = cislo % 2 == 0
return vysledek

print(sude_cislo(10)) # vypise: True
print(sude_cislo(11)) # vypise: False


Pro kratší zápis, není potřeba použít proměnnou vysledek, ale rovnou napsat příkaz return

def sude_cislo(cislo):
    return cislo % 2 == 0

print(sude_cislo(10)) # vypise: True
print(sude_cislo(11)) # vypise: False

def sude_cislo(cislo):
return cislo % 2 == 0

print(sude_cislo(10)) # vypise: True
print(sude_cislo(11)) # vypise: False


Máme seznam obsahující čísla. Pomocí funkce chceme zjistit, jestli se v seznamu vyskytuje alespoň jedno sudé číslo. Pokud ano, tak se vrátí hodnota True. Uvnitř funkce musí být cyklus, který projde všechny prvky v seznamu. Pokud číslo není sudé, tak bude cyklus pokračovat dál na další prvek.

def sude_v_seznamu(cisla):
    for i in cisla:
        if i % 2 == 0:
            return True
        else:
            pass

sude_v_seznamu([1, 3, 5])
sude_v_seznamu([2, 3, 5]) # True

def sude_v_seznamu(cisla):
for i in cisla:
if i % 2 == 0:
return True
else:
pass

sude_v_seznamu([1, 3, 5])
sude_v_seznamu([2, 3, 5]) # True


Uvedená funkce vrací pouze True, ale nevrací False, když tam není sudé číslo. Toto je potřeba vyřešit. Chyba by byla, kdyby za else místo příkazu pass jsme přidali return False

def sude_v_seznamu(cisla):
    for i in cisla:
        if i % 2 == 0:
            return True
        else:
            False # Chyba!

def sude_v_seznamu(cisla):
for i in cisla:
if i % 2 == 0:
return True
else:
False # Chyba!


Toto je chyba protože, funkce najde lichý prvek, vrátí False a cyklus skončí. Ale v seznamu může být stále nějaké sudé číslo, ke kterému se nedostalo, protože cyklus byl ukončen. Prohledávání tedy skončí hned na prvním lichém čísle a nejsou otestovány zbylé prvky v seznamu.

Příkaz return False musí být použit, pokud se celý seznam prošel a nenašlo se sudé číslo. V tomto případě se musí return False umístit až po skončení cyklu, tedy na konec.

def sude_v_seznamu(cisla):
    for i in cisla:
        if i % 2 == 0:
            return True
        else:
            pass

    return False # nenaslo sude cislo, tak vrati False

sude_v_seznamu([1, 3, 5]) # False
sude_v_seznamu([2, 3, 5]) # True

def sude_v_seznamu(cisla):
for i in cisla:
if i % 2 == 0:
return True
else:
pass

return False # nenaslo sude cislo, tak vrati False

sude_v_seznamu([1, 3, 5]) # False
sude_v_seznamu([2, 3, 5]) # True


Minulý příklad vracel informaci o tom, zda je v seznamu sudé číslo. Upravíme program tak, aby vrátil seznam čísel, která jsou sudá. Pokud sudé číslo nebude v seznamu, tak se vrátí prázdný seznam. Uvnitř funkce vytvoříme prázdný seznam, do kterého postupně budeme vkládat sudá čísla. Jakmile cyklus skončí, tak pomocí příkazu return, vrátíme vytvořený seznam.

def sude_v_seznamu(cisla):
    suda_cisla = [] # seznam

    for i in cisla:
        if i % 2 == 0:
            suda_cisla.append(i) # prida do seznamu
        else:
            pass
    return suda_cisla

print(sude_v_seznamu([1, 2, 8, 4, 5])) # vypise: [2, 8, 4]

suda_cisla = [] # seznam
