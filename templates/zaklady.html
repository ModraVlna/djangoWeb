Proměnné

Proměnné slouží k uchování hodnot. Proměnná se v Pythonu vytváří tak, že se napíše její jméno a přiřadí se k ní nějaká hodnota. Jsou určitá pravidla, která je potřeba dodržovat při vytváření proměnné. Název nesmí začínat číslem. Nesmí být nikde mezery, pro spojení více slov se často používá symbol podtržítko. Název proměnné nesmí obsahovat následující symboly :’”,<>/?|\()!@#$%^&~-+
Je zvykem psát název malými písmeny. Vyhnout se názvům, které mají speciální význam v Pythonu (list, str…)

cislo = 10
jmeno = "Tim"


V ukázce jsou dvě proměnné. První má název cislo, za názvem následuje symbol rovná se, které přiřadí hodnotu 10. Druhá proměnná má název jmeno, zase následuje symbol pro rovná se a poté řetězcová hodnota v uvozovkách.

Pro ověření, že proměnná cislo obsahuje hodnotu 10, tak vypíšeme tuto proměnnou na výstup. Pro výpis na výstup slouží funkce print(). Uvnitř kulatých závorek vložíme proměnnou cislo.

cislo = 10
jmeno = "Tim"

print(cislo) # vypise: 10


Stejným způsobem lze vypsat na výstup proměnnou jmeno.

cislo = 10
jmeno = "Tim"

print(jmeno) # vypise: Tim


Hodnoty, které jsme použili mají určitý datový typ. U proměnné cislo jsme zvolili hodnotu 10, což je celé číslo. U proměnné jmeno jsme zvolili hodnotu "Tim", což je řetězec. V Pythonu se celé číslo označuje jako int (od slova integer), řetězec se označuje jako str (od slova string). Jednotlivé datové typy budou probrány později. Pro zjištění jakého datové typu je proměnná, slouží funkce type()
Prvně použijeme funkci type() pro proměnnou cislo a poté pro proměnnou jmeno

cislo = 10
jmeno = "Tim"
print(type(cislo)) # vypise: < class 'int' >


cislo = 10
jmeno = "Tim"
print(type(jmeno)) # vypise: < class 'str' >


Python umožňuje přiřadit libovolný datový typ. Budeme mít tedy proměnnou x, obsahující celé číslo a poté hodnotu změníme na řetězec.

x = 5
print(x) # vypise: 5
x = "ahoj"
print(x) # vypise: ahoj


Datové typy

Při vytváření nějakého programu se pracuje s daty. Je potřeba, aby data byla nějakým způsobem reprezentována. Python nabízí různé druhy datových typů. Ukázka základních datových typu je možné vidět v následující tabulce.

Název Typ Popis
Celé číslo int 8   200   26
Desetinné číslo float 6.4   10.09   50.33
Řetězec str "Ahoj"  "Cokolada"  "1000"
Seznam list Uspořádaná posloupnost prvků: [5, "ahoj", 20.1]
Množina set Neuspořádané unikátní prvky: {"a", "b"}
N-tice tup Neměnitelná posloupnost prvků: (10, "c", 2)
Slovník dict Klíče a hodnoty {"klic": "hodnota"}

Jednotlivé datové typy budou popsány podrobněji pro lepší pochopení. Tabulka slouží jen jako ukázka.

Operátory

Operátory slouží pro provádění operací nad hodnotami a proměnnými.
Pro provádění základních matematických operací se používají aritmetické operátory, které jsou uvedeny v tabulce.

Operátor Název
+ Sčítání
- Odčítání
* Násobení
/ Dělení
% Modulo (zbytek po dělení)
** Mocnina

Pro vyzkoušení jednotlivých uvedených operátoru, vybereme si dvě hodnoty a výsledek vypíšeme na výstup.

print(10 + 5) # 15
print(10 - 5) # 5
print(10 * 5) # 50
print(10 / 5) # 2.0
print(10 % 5) # 0
print(10**5) # 100 000


Pro přiřazení hodnoty k proměnné používáme přiřazovací operátory. K proměnné x přiřadíme pomocí symbolu = hodnotu 5.

x = 5


Proměnná x tedy obsahuje hodnotu 5. V dalším kroku přičteme hodnotu 10 a x bude tedy rovno 15. Na levé straně bude x a na pravé straně bude původní x s hodnotou 5 a přičtení hodnoty 10.

x = 5
x = x + 10


Pro ověření aktuální hodnoty x, vypíšeme x na výstup

x = 5
x = x + 10
print(x) # vypise: 15


Uvedený zápis x = x + 10, lze zapsat i jinak. Symbol pro sčítání bude před symbolem rovná se. Původní x se na pravé straně se smaže. Tento zápis je kratší.

x = 5
x += 10
print(x) # vypise: 15


Podobně lze použít operátory pro odčítaní, násobení atd.

Pro srovnání hodnot jsou k dispozici tyto operátory:

Operátor Název
== Je rovno
!= Není rovno
> Je větší než
< Je menší než
>= Je větší nebo rovno
<= Je menší nebo rovno

Pro vyzkoušení těchto operátorů budeme mít dvě proměnné x, y.
x bude rovno 5, y bude rovno 10.
Budeme tyto dvě hodnoty srovnávat. Pokud je tvrzení pravdivé, na výstup se vypíše True, v opačném případě se vypíše False.

x = 5
y = 10
print(x == x) # True
print(x == y) # False
print(x != y) # True


Podobně porovnáme pomocí dalších operátorů.

x = 5
y = 10
print(x > x) # False
print(x < y) # True
print(x >= y) # False
print(x <= y) # True


Logické operátory se používají při vyhodnocování nějakých podmínek.

Operátor Název
and Vrací True, když obě podmínky jsou pravdivé
or Vrací True, když alespoň jedna podmínka je pravdivá
not Vrací opak výsledku, vrátí False, pokud je výsledek True

x = 5
y = 10
print(x > 2 and y > 5) # True
print(x > 20 or y > 2) # True
print(not(x > 2 and y > 5)) # False


Čísla

V Pythonu jsou tři typy čísel. Datový typ int, float a complex. Celé číslo je int, desetinné číslo je float a komplexní číslo je complex.

a = 2 # int
b = 3.6 # float
c = 8j # complex


Datový typ int jsou celá čísla, mohou být kladná nebo záporná, bez desetinné tečky.
Datový typ float jsou desetinná čísla, mohou být kladná nebo záporná, s desetinnou tečkou.
Datový typ komplex se píšou s písmenem j.

Python umožňuje převést jeden datový typ na druhý. Je to pomocí funkcí int(), float() nebo complex(). Komplexní čísla nelze převést na jiný datový typ. Název funkce popisuje, na který datový typ chceme číslo převést. Převáděné číslo se vkládá do závorky. Převedeme tedy číslo datového typu int na datový typ float.

a = 2 # int
x = float(a) # z int na float
print(x) # vypise: 2.0


Řetězce

Řetězec je posloupnost znaků, který se zapisuje pomocí jednoduchých nebo dvojitých uvozovek.

a = 'Ahoj'
b = "Ahoj"


Python umožnuje mít součástí řetězce uvozovku.

a = "Tento retez obsahuje ' uvozovku "


Jelikož řetězec je uspořádána posloupnost znaků, můžeme nad ním provádět operace jako indexování (anglicky indexing) a řezání (anglicky slicing).

Každý znak v řetězci má svoji pozici, nebo-li svůj index. Tento index je vyjádřen číslem. Počáteční znak má hodnotu 0. Poslední znak má hodnotu rovno počtu znaků, méně 1.

a = "ahoj"
# a = 0, h = 1, o = 2, j = 3


Máme řetězec "ahoj", který je složen ze 4 znaků. První znak je písmeno a, který má index 0.
Python umožňuje záporné indexování. Index s hodnotou -1 představuje poslední prvek v řetězci. Hodnota -2 je pro předposlední prvek atd.

Máme řetězec a budeme chtít vypsat pouze první znak. Víme, že první znak je na pozici 0. Použijeme k tomu metodu print(). Zadáme název řetězce, za ním budou následovat hranaté závorky. Uvnitř těchto hranatých závorek bude pozice našeho znaku.

slovo = "ahoj"
print(slovo[0]) # vypise: a


V případě, že chceme vypsat poslední prvek a nevíme délku řetězce, použijeme záporný index.

slovo = "ahoj"
print(slovo[-1]) # vypise: j


Python umožnuje vypsat i část nějakého řetězce. Budeme mít řetězec automobil. Prvním úkolem je vypsat pouze slovo mobil.
Zjistíme index znaku m. V tomto případě je m na pozici 4, protože číslování je od nuly. Do hranatých závorek dáme číslo 4, poté bude následovat dvojtečka, které představuje zbytek znaků.

slovo = "automobil"
print(slovo[4:]) # vypise: mobil


Druhým úkolem je vypsat slovo auto. Rozdíl bude v tom, že dvojtečka bude před číslem. Znamená to, že začínáme od prvního písmene (s indexem 0).
Poslední písmeno ve slově auto je o, které je na pozici 3. Ale v závorce bude hodnota 4. Znamená to, že budou vybrána všechna písmena od počátku po index 4, ale index 4 nebude zahrnut.

slovo = "automobil"
print(slovo[:4]) # vypise: mobil


Také lze vybrat znaky podle rozsahu. Vybereme první dva znaky z řetězce. V závorce bude číslo představující začátek, poté dvojtečka a následně číslo, které představuje konec, ale není zahrnuto.

slovo = "automobil"
print(slovo[0:2]) # vypise: au


Také je možné určit kolik znaků se má přeskočit. Toto je pomocí třetího parametru, kde číslo představuje o kolik se má přeskočit.
Budeme chtít vypsat každé druhé slovo v řetězci. Chceme tedy vypsat každý druhý znak. Použijeme tedy hodnotu 2, která bude na konci závorek, jelikož používáme třetí parametr, tak budou dvojtečky dvě.

slovo = "automobil"
print(slovo[::2]) # vypise: atmbl


Dalším úkolem je na výstup vypsat řetězec, od druhého indexu, po index 5 (který není zahrnut), se skokem 2

slovo = "automobil"
print(slovo[2:5:2]) # vypise: tm


Podobně lze vypsat na výstup řetězec pozpátku. Použije se k tomu index se zápornou hodnotu.
Od začátku až po konec řetězce se skokem -1.

slovo = "automobil"
print(slovo[::-1]) # vypise: libomotua


V následující tabulce jsou typické metody, které lze použít s řetězci.

Metody Popis
.capitalize() Převede počáteční písmeno na velké
.count() Vrátí počet výskytu určité hodnoty v řetězci
.find() Vrátí hledanou hodnotu uvnitř řetězce
.index() Vrátí index dané hodnoty v řetězci
.lower() Převede všechny písmena na malá v řetězci
.upper() Převede všechny písmena na velká v řetězci

Boolean

Boolean může být True nebo False. True představuje pravdivou hodnotu. False představuje nepravdivou hodnotu. V Pythonu se True nebo False píše velkým počátečním písmenem.
Často se boolean používá pro vyhodnocení nějaké podmínky. Např. porovnání dvou hodnot.

print(5>2) # vypise: True


Casting

Casting v Pythonu je přetypování, kdy máme datový typ a chceme jiný datový typ. Pro casting lze v použít funkce int(), float(), str(). Uvnitř závorek funkcí budeme vkládat hodnoty, které chceme, aby byly daného typu.

a = int(5) # z int na int
b = int(1.3) # z float na int
c = int("10") # z string na int
print(a) # vypise: 5
print(b) # vypise: 1
print(c) # vypise: 10


Komentáře

Komentáře se používají k vysvětlení nebo k popisu kódu. Komentáře se vkládají ke kódu, který chceme popsat. Komentář začíná symbolem # a za ním je libovolný text.

x = 10
y = 5
# scitani dvou cisel
print(x+y)


Pro napsání komentáře na několik řádku, lze na každém řádku mít symbol #

# Tento komentar
# je napsan
# na vice radku
x = 10


Kromě symbolu #, lze také použít tři uvozovky.

"""
Tento komentar
je napsan
na vice radku
"""
x = 10

print(a+b) # 15

b = 10
print(a==b) # True

x = 5
x += 20
print(x) # 25

slovo = "dnes"
print(slovo)

    Vypište na výstup první písmeno.

ret = "drahokam"

    Vypište na výstup poslední písmeno pomocí záporného indexování.

ret = "drahokam"

  • 7. Máte proměnnou ret = "drahokam"
    Vypište na výstup první tři znaky řetězce.

ret = "drahokam"
print(ret[:3]) # dra

    Vypište na výstup každý druhý znak.

ret = "drahokam"
print(ret[::2]) # daoa

    Vypište na výstup znaky řetězce v obráceném pořadí.

ret = "drahokam"
print(ret[::-1]) # makohard



  • 10. Vytvořte dvě proměnné, které budou obsahovat řetězec.
    Porovnejte tyto dva řetězce a na výstup vypište, jestli jsou stejné.

b = "pondeli"
