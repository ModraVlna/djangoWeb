Třídy a objekty

Objektově orientované programovaní (anglicky Object Oriented Progamming) umožnujě vytvářet vlastní objekty, které mají metody a datové atributy.

Třída

Pro vytvoření vlastního objektu je potřeba definovat třídu. Uvnitř třídy definujeme atributy a metody (funkce). K vytvoření třídy se použije klíčové slovo class a za ním, námi zvolaný název dané třídy začínající velkým počátečním písmenem. Za názvem je dvojtečka.

class TotoJeTrida:


Vytvoříme třídu s názvem Pes. Tato třída bude obsahovat metodu pozdrav(), které po zavolání vypíše něco na výstup.

class Pes:
    def pozdrav(self):
        print("Baf")


Je tedy vytvořená jednoduchá třída, která obsahuje jednu metodu. Abychom mohli přistoupit k této metodě, tak musíme vytvořit objekt dané třídy. Objekty, které patří do dané třídy se také označují jako instance této třídy.
Pro vytvoření objektu patřící do třídy Pes, je potřeba zvolit jeho název a specifikovat danou třídu.

muj_pes = Pes()


Nyní lze přistoupit k metodě pozdrav pomocí tečkové notace.

muj_pes.pozdrav() # vypise: Baf


Celý kod vypadá následovně:

class Pes:
    def pozdrav(self):
        print("Baf")

muj_pes = Pes()
muj_pes.pozdrav() # vypise: Baf


Datové atributy a metody objektu

Pes může mít nějaké vlastnosti, jako jsou např. věk, plemeno nebo barva.
Vytvoříme třídu Pes, která bude mít atributy vek a plemeno.
Uvnitř třídy vytvoříme metodu s názvem __init__
__init__ je speciální metoda, která je zavolána, když je vytvořena instance dané třídy. U každé metody dané třídy je prvním parametrem self. Parametr self je odkaz na daný objekt dané třídy a používá se k přístupu proměnných, které patří třídě. Parametr self se může jmenovat i jinak, ale vždy musí být jako první parametr každé metody.

Metoda __init__ bude mít tedy parametr self a dále vek a plemeno.

class Pes:
    def __init__(self, vek, plemeno):


Tělo metody bude obsahovat: self.vek = vek a self.plemeno = plemeno

class Pes:
    def __init__(self, vek, plemeno):
        self.vek = vek
        self.plemeno = plemeno


Metoda __init__ je zavolána automaticky, když je vytvořena instance třídy. Klíčové slovo self představuje instanci objektu samotného. Potom se očekává, že zadáme věk a plemeno nějakého psa. Tento věk a toto plemeno přiřadíme k self.vek a self.plemeno

Poté se vytvoří objekt, kde specifikujeme daný věk a plemeno psa. Na výstup vypíšeme plemeno psa.

class Pes:
    def __init__(self, vek, plemeno):
        self.vek = vek
        self.plemeno = plemeno

muj_pes = Pes(10, "Nemecky ovcak")
print(muj_pes.plemeno) #vypise: Nemecky ovcak


Na pořadí argumentů záleží. Kromě uvedeného způsobu lze argumenty zapsat takto:

muj_pes = Pes(vek = 10, plemeno = "Nemecky ovcak")
print(muj_pes.plemeno) #vypise: Nemecky ovcak


Vytvoříme třídu s názvem Clovek. Tato třída bude mít atributy představující jméno, věk a informaci o tom, jestli pracuje. Zda pracuje bude rozhodnuto podle True nebo False. Dále bude uvnitř třídy metoda, které sdělí informace o cloveku. Na výstup vypíše jméno a věk. Pokud bude osoba pracovat, tak se vypíše, že pracuje. V opačném případě se vypíše, že nepracuje. Toto může provést pomocí if a else.

class Clovek:
    def __init__(self, jmeno, vek, prace):
        self.jmeno = jmeno
        self.vek = vek
        self.prace = prace

    def informace(self):
        print("Jmenuji se " + self.jmeno)
        print("Je mi", self.vek, "let")
        if self.prace == True:
            print("Pracuji")
        else:
            print("Nepracuji")

osoba = Clovek("Gustav", 30, True)
osoba.informace()
#Jmenuji se Gustav
#Je mi 30 let
#Pracuji


Dědičnost

Dědičnost umožnuje vytvořit třídu, která bude dědit vlastnosti a metody z jiné třídy. Třída, z které se dědí je rodič. Třída, které dědí od rodiče se nazývá potomek.

Budeme mít třídu Zvire. Potomkem této třídy bude třída Pes a třída Kocka. Abychom mohli dědit vlastnosti ze třídy Zvire, musíme její název uvézt v závorce za názvem potomka.

class Zvire:
    pass

class Pes(Zvire):
    pass

class Kocka(Zvire):
    pass


Připravíme si tři třídy. Vozidlo, který bude rodič a potomky Automobil a NakladniAutomobil.

class Vozidlo:
    pass
class Automobil(Vozidlo):
    pass
class NakladniAutomobil(Vozidlo):
    pass


Třída Vozidlo bude mít atributy palivo, barva a cena. Dále bude mít tři metody, které vrátí atribut palivo, barvu nebo cenu.

class Vozidlo:
    def __init__(self, palivo, barva, cena):
        self.palivo = palivo
        self.barva = barva
        self.cena = cena

    def vrat_palivo(self):
        return self.palivo

    def vrat_barvu(self):
        return self.barva

    def vrat_cenu(self):
        return self.cena


Třída Automobil bude mít další atribut rychlost. V metodě __init__ bude použito klíčové slovo super() pro dědění z třídy Vozidlo. Dále bude třída Automobil mít metodu nastartuj(), která vypíše na výstup, že se auto nastartovalo.

class Automobil(Vozidlo):
    def __init__(self, palivo, barva, cena, rychlost):
        super().__init__(palivo, barva, cena)
        self.rychlost = rychlost

    def nastartuj(self):
        print("Auto nastartovano")


Podobně bude i třída NakladniAutombil. Ta nebude mít rychlost, místo toho bude mít atribut kola, který představuje počet kol na nákladním automobilu.

class NakladniAutomobil(Vozidlo):
    def __init__(self, palivo, barva, cena, kola):
        super().__init__(palivo, barva, cena)
        self.kola = kola
    def nastartuj(self):
        print("Nakladni automobil nastartovan")


Nyní vytvoříme dva objekty, s názvem skoda a iveco. skoda bude instance třídy Automobil, iveco bude instance třídy NakladniAutomobil. U obou instancí zavoláme metodu nastartuj() a také metodu vrat_palivo().

skoda = Automobil(40, "bila", 400000, 200)
iveco = NakladniAutomobil(150, "cerna", 1500000, 12)

skoda.nastartuj() # vypise: Auto nastartovano
print(skoda.vrat_palivo()) #vypise: 40

iveco.nastartuj() # vypise: Nakladni automobil nastartovan
print(iveco.vrat_palivo()) # vypise: 150


Polymorfismus

Dva objekty různých tříd mohou mít stejný název metody. Polymorfismus v Pythonu umožnuje metodu zavolat ze stejného místa pro různé objekty. Budeme mít třídu Pes a Kocka. Atributem bude jmeno. Obě třídy budou mít metodu s názvem mluv, která vrací jméno.

class Pes:
    def __init__(self, jmeno):
        self.jmeno = jmeno

    def mluv(self):
        return self.jmeno + " rekl baf"


class Kocka:
    def __init__(self, jmeno):
        self.jmeno = jmeno

    def mluv(self):
        return self.jmeno + " rekla mnau"


Dále vytvoříme dvě instance

muj_pes = Pes("Rex")
moje_kocka = Kocka("Minda")


U obou instancí můžeme zavolat metodu mluv(). Jelikož název metod je stejný můžeme metodu zavolat pro obě instance zároveň.
Toto lze provést pomocí for cyklu, kde do hranatých závorek dáme dané instance.

for zvire in [muj_pes, moje_kocka]:
    print(zvire.mluv()) # vypise: Rex rekl baf Minda rekla mnau


Také lze použít vlastní metodu, kde parametrem bude daná instance.

def mluveni(zvire):
    print(zvire.mluv())

mluveni(muj_pes) # vypise: Rex rekl baf


Abstraktní třída

U abstraktní třídy se neočekává, že se bude vytvářet její instance. Je používána s dědičností. Budeme mít abstraktní třídu Zvire. Potomkem této třídy bude třída Pes. Pomocí metody mluv() se vrátí jméno.
Pokud by se vytvořila instance třídy Zvire a použila by se metoda mluv, tak se vyhodí chyba.

class Zvire:
    def __init__(self, jmeno):
        self.jmeno = jmeno

    def mluv(self):
        raise NotImplementedError("Potomek neni implementovan")

class Pes(Zvire):
    def mluv(self):
        return self.jmeno

zviratko = Zvire("zvire")
print(zviratko.mluv()) # vyhodi chybu

muj_pes = Pes("Rex")
print(muj_pes.mluv()) # vypise: Rex


Speciální metody (Magic/Dunder)

Budeme mít třídu Kniha, která má atributy nazev, autor a strany. Při použití funkce print() pro výpis instance nedostaneme informace o atributech. Dostaneme pouze informaci o tom, kde v paměti je uložen daný objekt.
Toto lze vyřešit pomocí speciální metody s názvem __str__, která bude uvnitř třídy.
Tato metoda vrátí informaci o názvu a autora dané knihy při její zavolání.

class Kniha:
    def __init__(self, nazev, autor, strany):
        self.nazev = nazev
        self.autor = autor
        self.strany = strany

    def __str__(self):
        return f"{self.nazev} od {self.autor}"

moje_kniha = Kniha("Pyhon knina", "Gustav", 200)
print(moje_kniha) # vypise Python kniha od Gustav


Těchto speciálních metod je více. Chceme zjistit počet stran knihy. K tomuto použijeme speciální metodu __len__(). Dále chceme mít nějakou informaci o tom, že jsme smazali objektu. K tomuto použijeme speciální metodu __del__()

class Kniha:
    def __init__(self, nazev, autor, strany):
        self.nazev = nazev
        self.autor = autor
        self.strany = strany

    def __str__(self):
        return f"{self.nazev} od {self.autor}"

    def __len__(self):
        return self.strany

    def __del__(self):
        print("Instance knihy byla smazana")

moje_kniha = Kniha("Pyhon knina", "Gustav", 200)
print(moje_kniha) # vypise Python kniha od Gustav
print(len(moje_kniha)) # vypise: 200
del moje_kniha # vypise: Instance knihy byla smazana


Příklady


  • 1. Napište třídu Auto, které neobsahuje žádné metody a atributy.

class Auto:
    pass



  • 2. Napište třídu Produkt. Tato třídá bude mít atributy nazev a cena.
    Třída bude mít metodu __init__()
    Dále bude mít dvě metody. První metoda bude vracet atribut nazev. Druhá metoda bude vracet atribut cena.

class Produkt:
    def __init__(self, nazev, cena):
        self.nazev = nazev
        self.cena = cena

    def dej_nazev(self):
        return self.nazev

    def dej_cenu(self):
        return self.cena



  • 3. Napište třídu Ucet, která bude sloužit pro bankovní účet.
    Tato třída bude mít atribut uzivatel a penize.
    Dále budou zde dvě metody pro vklad a výběr peněz.
    Uživatel nemůže vybrat z účtu víc peněz než má k dispozici.

class Ucet:
    def __init__(self, uzivatel, penize):
    self.uzivatel = uzivatel
    self.penize = penize

    def vklad(self, mnozstvi):
    self.penize += mnozstvi

    return self.penize

    def vyber(self, mnozstvi):
        if self.penize >= mnozstvi:
            self.penize -= mnozstvi
        else:
            print("Nedostatek penez na uctu")

        return self.penize
