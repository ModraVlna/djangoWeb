Instalace

Windows

Otevřeme terminál.
Pro používání frameworku Django musíme mít nainstalovaný Python. Musíme ověřit, že je Python nainstalovaný.

python3 -V


Pokud není nainstalovaný, tak lze ho stáhnout z oficiálních stránek. Je doporučeno stáhnout verzi Python 3.8+

Na libovolném místě vytvoříme soubor, např. djangoprojekt
Poté otevřeme terminál a budeme se nacházet uvnitř djnagoprojektu.
Provedeme příkaz pro vytvoření virtuálního prostředí

python -m venv env


Po vytvoření virtuálního prostředí, ho musíme aktivovat.

env\Scripts\activate


Nyní ve virtuálním prostředí nainstalujeme Django.

pip install django


Poté můžeme vytvořit náš vlastní Django projekt. Vybereme si libovolný název, např. django_aplikace

django-admin startproject django_aplikace


Máme vytvořený Django projekt s názvem django_aplikace
Přepneme se do toho projektu

cd django_aplikace


macOS

Otevřeme terminál, ve kterém budeme pracovat.
Pro používání frameworku Django musíme mít nainstalovaný Python. Musíme ověřit, že je Python nainstalovaný.

python3 -V


Pokud není nainstalovaný, tak lze ho stáhnout z oficiálních stránek. Je doporučeno stáhnout verzi Python 3.8+
Poté se musí nainstalovat pip a také pipenv

python3.8 -m pip install pip –upgrade
python3.8 -m pip install pipenv


Pro ověření, že se pipenv nainstalovalo, použijeme příkaz

pipenv


Pomocí pipenv vytvoříme virtuální prostředí.
V terminálu si vytvořte nový adresář a pojmenujte ho např. dev

mkdir dev


V adresáři dev, vytvoříme nový adresář, který libovolně pojmenujeme, např. djangoprojekt.

mkdir djangoprojekt


Nový vytvořený adresář djangoprojekt musí být na začátku prázdný.
V tomto adresáři provedeme příkaz

virtualenv -p python3 .


source bin/activate


Nyní nainstalujeme Django. Pro nainstalovaní aktuální verze je příkaz

pip install django


Konkrétní verze lze najít na oficiálních stránkách Django. Instalování určité verzi lze provést takto:

pip install django==3.0.4


Po úspěšné instalaci zavřeme terminál.

Linux

Otevřeme terminál.
Pro používání frameworku Django musíme mít nainstalovaný Python.

sudo apt-get install python3


Poté nainstalujeme pip

sudo apt-get install python3-pip


Nainstalujeme Django

pip3 install Django
sudo apt-get install python3-django


Vytvoříme adresář, ve kterém bude náš Django projekt, např. django_projekt

mkdir django_projekt


Uvnitř tohoto adresáře se vytvoří náš Django projekt.

sudo django-admin startproject django_aplikace
