Podmíněný příkaz

Podmíněný příkaz se používá, když máme program, který vykoná daný příkaz v závislosti na nějaké podmínce. Při práci s podmíněním příkazem využíváme klíčová slova if, elif a else.

If, elif, else

Při vytvoření jednoduchého podmíněné příkazu použijeme if. Poté je podmínka, kterou uvedeme, následuje dvojtečka. Příkazy na novém řádku jsou odsazeny. Odsazení se dělá pomocí tabulátoru (na klávesnici tab).

if nejaka_podminka:
    # proved prikaz


Máme číslo a chceme zjistit, jestli toto číslo je vetší než hodnota 10. Pokud ano, tak se vypíše zpráva. Porovnávají se tedy dvě hodnoty. Nejprve se napíše klíčové slovo if. Poté bude podmínka. Podmínka je, že naše číslo je větší než 10. Na konci bude dvojtečka. Na dalším řádku bude příkaz, který vypíše informace. Tento příkaz na novém řádku musí být odsazen pomocí tabulátoru.

cislo = 50
if cislo > 10:
    print(cislo, "je vetsi nez 10") # vypise: 50 je vetsi nez 10


Změníme naše číslo z hodnoty 50, na hodnotu 3. Porovná se tedy, jestli hodnota 3 je větší než 10. Uvedený podmíněný příkaz vypíše informaci pouze tehdy, pokud je číslo větší. Chceme, ale aby vypsalo zprávu, že hodnota je menší. Proto se použije klíčové slovo else. Číslo je tedy buď větší nebo menší než 10.

cislo = 3
if cislo > 10:
    print(cislo, "je vetsi nez 10")
else:
    print(cislo, "je mensi nez 10") # vypise: 3 je mensi nez 10


Máme tedy program, který dokáže rozpoznat, zda nějaké číslo je vetší nebo menší. Ale může se vyskytnout situace, že čísla jsou si rovna. Pokud je potřeba mít více podmínek, tak lze přidat další podmínku pomocí klíčového slova elif. Použijeme elif pro porovnání, zda naše číslo je rovno 10.

cislo = 10
if cislo > 10:
    print(cislo, "je vetsi nez 10")
elif cislo == 10:
    print(cislo, "je rovno 10") # vypise: 10 je rovno 10
else:
    print(cislo, "je mensi nez 10")


Vytvoříme program, který podle zadaného místa budovy vypíše její adresu. Vytvoří se proměnná s názvem misto, která bude obsahovat název budovy. Pomocí if, elif a else se bude místo porovnávat s hodnotou v podmínce. Když bude proměnná obsahovat budovu Obchod, tak program vypíše adresu budovy. Pokud se místo nebude shodovat z žádnou z uvedených možnosti, tak bude vypsáno, že adresa je není známa.

misto = "Obchod"

if misto == "Banka":
    print("Banka je na adrese Nova 20")
elif misto == "Obchod":
    print("Obchod je na adrese Hlavni 5")
elif misto == "Restaurace":
    print("Restaurace je na adrese Nadrazni 12")
else:
    print(misto, "nema znamou adresu")


Cyklus

Cyklus se používá, když chceme část kódu provést několikrát. Máme tedy nějakou činnost, která se bude opakovat. Cyklus umožní vykonat příkazy vícekrát. Python poskytuje for cyklus a while cyklus.

Cyklus for

U mnoha objektů v Pythonu lze iterovat. Např. se dá iterovat každý znak v řetězci, každý prvek v seznamu atd. Připravíme si seznam, který bude obsahovat čísla. Tyto čísla pomocí for cyklu vypíšeme.

cisla = [1, 2, 3, 4, 5]


Pro každý prvek v seznamu, vypíšeme daný prvek. Použijeme k tomu klíčové slovo for a in.

for prvek in cisla:
    print(prvek) # vypise: 1 2 3 4 5


Podobně se dá vypsat jednotlivé znaky nějakého řetězce.

retezec = "automobil"
for i in retezec:
    print(i) # vypise: a u t o m o b i l


Máme seznam obsahující čísla a chceme vypsat jejich druhou mocninu. Mocnina se zapisuje v Pythonu pomocí dvou hvězdiček.

cisla = [5, 2, 4]
for i in cisla:
    print(i**2) # vypise: 25 4 16


Uvnitř for cyklu můžeme provádět další operace. Např. chceme vypsat pouze čísla, které jsou sudá. Číslo je sudé, jestliže jeho zbytek po dělení je rovno nule. Pro zjištění zbytku je k dispozici mod, který se označuje symbolem %. Pokud zbytek po dělení čísla je rovno nule, tak to znamená, že číslo je sudé a vypíšeme ho.

cisla = [1, 2, 3, 4, 5, 6]
for i in cisla:
    if i % 2 == 0:
        print(i) # vypise: 2 4 6


range()

Funkce range() vytvoří posloupnost hodnot. Počet hodnot se vkládá do závorky. Posloupnost začíná číslem 0 a zvyšuje se o jedničku. Použijeme funkci range() a vypíšeme hodnoty na výstup pomocí for cyklu.

hodnoty = range(3)
for i in hodnoty:
    print(i) # vypise: 0 1 2


Lze stanovit od jakého čísla má posloupnost začínat.

hodnoty = range(3, 6)
for i in hodnoty:
    print(i) # vypise: 3 4 5


Třetím nepovinným parametrem je o kolik se mají čísla zvyšovat. Např. místo jedničky to bude o dva.

hodnoty = range(3, 10, 2)
for i in hodnoty:
    print(i) # vypise: 3 5 7 9


Cyklus while

While je druh cyklu, který provádí nějaké příkazy, když je podmínka pravdivá. Např. zatímco můj bazén není plný, tak ho napouštěj vodou. Pro použití while cyklu se musí uvést klíčové slovo while, potom následuje podmínka a poté nějaké činnost.

Budeme mít číslo, které budeme vypisovat a zvyšovat jeho hodnotu o jedničku. Zastavíme výpis a přičítání, když bude naše číslo menší jak 5.

cislo = 1
while cislo < 5:
    print(cislo) # vypise: 1 2 3 4
    cislo = cislo + 1 # pricte k cislu jednicku


cislo představuje proměnnou s hodnotou jedna na začátku. Podmínka je cislo < 5. Poté následuje dvojtečka, za ní je tělo cyklu.

while cyklem lze použít příkaz else. Kód uvnitř else se provede, když bude podmínka nepravdivá.

cislo = 1
while cislo < 5:
    print(cislo) # vypise: 1 2 3 4
    cislo = cislo + 1 # pricte k cislu jednicku
else:
    print("Cyklus skoncil")


Break, continue, pass

S cyklem lze také použít tři příkazy, tím je break, continue a pass.

Příkaz break (zastavit) umožnuje zastavit cyklus, i když je podmínka stále pravdivá. Provedeme příkazy v těle pouze jednou.

cislo = 1
while cislo < 5:
    print(cislo) # vypise: 1
    cislo = cislo + 1
    break


Zastavíme cyklus, když číslo bude rovno hodnotě 3.

cislo = 1
while cislo < 5:
    print(cislo) # vypise: 1 2
    cislo = cislo + 1
    if cislo == 3:
        break


continue je příkaz, který se používá pro zastavení aktuální iterace a pokračování v další. Budeme mít řetězec, máme za úkol vypsat tento řetězec, ale vynechat písmena o, která se vyskytují v řetězci. K tomuto použijeme příkaz continue.

retezec = "automobil"
for i in retezec:
    if i == "o":
        continue
    print(i) # vypise: a u t m b i l


pass je příkaz, který se používá, když je nějaký cyklus, ale tělo tohoto cyklu je prázdný. Pokud se v Pythonu vytvoří cyklus, který nic neobsahuje, tak to vyhodí syntaktickou chybu. Proto se použije příkaz pass, aby nedošlo k žádné chybě. Příkaz pass se např. používá, když si chceme připravit cyklus na budoucí použití.

retezec = "automobil"
for i in retezec:
    # vyhodi to syntaktickou chybu


Použití příkazu pass

retezec = "automobil"
for i in retezec:
    pass # bez chyby


Cyklus obsahující další cyklus

Python umožnuje použít cyklus, který obsahuje další cyklus. Budeme mít dva seznamy. První seznam obsahuje barvy a druhý seznam obsahuje ovoce. Úkolem je vypsat každou barvu s každým ovocem společně. K tomuto použijeme dva for cykly. První for cyklus bude procházet prvky v seznamu barvy. Druhý cyklus bude procházet prvky v seznamu ovoce. Potom se tyto prvky vypíší na výstup.

barva = ["cervena", "zluta"]
ovoce = ["jablko", "hruska"]

for i in barva:
    for j in ovoce:
        print(i, j) # i obsahuje prvek z barvy, j obsahuje prvek z ovoce
# cervena jablko, cervena hruska, zluta jablko, zluta hruska
