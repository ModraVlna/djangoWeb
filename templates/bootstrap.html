Bootstrap a static

Bootstrap v Django

Bootstrap je framework, pomocí kterého můžeme stylovat webovou aplikaci použitím šablon pro ikony, texty, tlačítka, navigační menu a další. Bootstrap používá CSS, Javascript, jQuery a je dostupný zdarma.

Abychom mohli využívat předem připravené styly od Bootstrap, tak musíme přidat nějaký kód uvnitř našeho projektu.
Otevřeme naši základní šablonu, ze které ostatní šablony dědí. Tato šablona je base.html Na úplném začátku přidáme tag

<!doctype html>


Uvnitř <head></head> bude meta tag pro kódování v UTF-8 a druhý meta tag, aby se webová aplikace správně zobrazovala i na mobilním zařízením.

<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1">


Dále bude následovat uvnitř <head></head> stylesheet pro načtení CSS.

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-BmbxuPwQa2lc/FVzBcNJ7UAyJxM6wuqIj61tLrc4wSX0szH/Ev+nYRRuWlolflfl" crossorigin="anonymous">


Na úplném konci <body></body> přidáme skript, který slouží pro různé animace tlačítek atd.

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta2/dist/js/bootstrap.bundle.min.js" integrity="sha384-b5kHyXgcpbZJO/tY9Ul7kGkf1S0CWuKcCD38l8YkeH8z8QjE0GmW1gYU5S9FOnJ0" crossorigin="anonymous"></script>

Použití Bootstrap

Veškeré nabízené styly jsou k dispozici na oficiálních stránkách Bootstrap.
Pro otestování vložíme tlačítko z Bootstrap do našeho projektu.
Otevřeme si šablonu, např. home.html, kam budeme chtít tlačítko přidat.
Na oficiálních stránkách jsou v položce Buttons, dostupná tlačítka různých velikostí a barev.

Vložíme do šablony dvě tlačítka, jedno bude zelené, které bude sloužit ke koupi výrobku. Druhé tlačítko bude červené, pro prodej výrobku. Kód pro tlačítka vložíme uvnitř <body></body>

<button type="button" class="btn btn-success">Koupit</button>
<button type="button" class="btn btn-danger">Prodat</button>


Static soubor

Static soubor slouží k ukládání CSS, Javascript kódu a obrázků. Pomocí static souboru můžeme rozdělit tyto kódy a obrázky od Pythonu a šablon v rámci projektu. Např. je webová aplikace, která má mnoho obrázků. Je lepší mít všechny obrázky na jednom místě pro lepší přehlednost a přístup.

Vytvoření static souboru

Uvnitř našeho projektu vytvoříme nový soubor a pojmenujeme ho static. Uvnitř static vytvoříme tři další soubory. Soubory se budou jmenovat css, js, images
Uvnitř css souboru vytvoříme soubor zvaný main.css
Soubor main.css bude obsahovat CSS kód.
CSS kód budeme tedy vkládat do css.
Javascript kód budeme vkládat do js.
Obrázky budeme vkládat do images.

V settings.py musíme nastavit cesty k těmto souborům, abychom je mohli používat s Django.
Na úplném konci přidáme settings.py přidáme toto:

STATICFILES_DIRS =[ Path.joinpath(BASE_DIR , 'static'), ]


Dále otevřeme náš base.html. Musíme sem vložit odkaz na náš CSS kód.
Vložíme to uvnitř <head></head>

<link rel="stylesheet" type="text/css" href="{% static '/css/main.css' %}">


Pro obrázky musíme v settings.py přidat

MEDIA_URL = /'images'/


Vložíme do images obrázek, bude mít název např. obr1.png

Uvnitř base.html na úplném začátku napíšeme kód pro načtení static souboru.

{% load static %}


Obrázek přidáme takto:

<img src="{% static 'images/obr1.png' %}">
